# Job Ad Gender-Debiasing Tool
IT/SE Job Advertisement Gender Bias Detection Tool - FIT4003 Project 2022 - Team 24

Repo for our FIT 4003 Final year project. Creating a tool that can determine whether or not gender inclusivity recommendations have been met using natural Language processing and machine learning

## Installation
Use the package manager [pip](https://pip.pypa.io/en/stable/) to install requirements.

```bash
pip install - requirements.txt
```

## Running the tool
Once requirements have been installed, run the following script to run the Flask application on your local host.

```python
python main.py

```

## How to use the tool
1. Press the Try It button to start using the tool 
2. Copy and paste your job advertisement into the text box, and press Submit 
3. Your job advertisement will be analysed using our machine learning model and recommendations will be given
4. Edit your job advertisement according to the recommendations to make it more inclusive!
